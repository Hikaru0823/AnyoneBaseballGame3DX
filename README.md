# AnyoneBaseballGame3DX  
**重度障害児でも遊べる野球盤ゲーム**  

## 🎯 概要  
本プロジェクトは、視線入力・スイッチ入力・キーボードなど多様な操作方式に対応し、**誰でもプレイできる3D野球ゲーム**を目指した作品です。  
アクセシビリティと操作感、ネットワーク対戦機能にこだわり、児童やその家族にも配慮した設計になっています。

## ✅ 主要機能 / 特徴
- 多種多様なモード選択  
- 視線入力による操作 （別途アプリが必要）
- スイッチ入力対応  
- キーボード入力対応  
- アクセシビリティ対応 UI（大きめボタン、マウス滞留でボタン押下）  
- ネットワーク対戦機能（チーム戦）

## 🛠️ 環境 / 実行方法  
- Unity バージョン：6000.0.36f1
- 動かし方：リポジトリをクローン → Unity で `Assets/Scenes/TitleScene.unity` を開く → プレイボタン  
- 視線入力はTobii社のEyeTracker、EyeMoTMouse.exeが別途必要。

## 💡 技術的・設計上の工夫点 / 背景  
- ユーザーテスト及び実機テストを通じた入力方式の調整  
- アクセシビリティを重視した UI／操作設計  
- “誰でも遊べる”という目的のための仕様や設計意図(バリアフリーモードは児の成功体験のため、振れば当たる仕様)
- オンライン対戦機能実装のためのPhoton2SDKの活用
- 特に力を入れたスクリプト（オンライン対戦機能）　→　`Assets/Scripts/Network/PlayerRegistry.cs` 、`Assets/Scripts/Network/PlayerObject.cs` 、`Assets/Scripts/Network/GameState.cs`
